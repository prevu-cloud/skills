# Prevu Cloud Skills

Skills for coding agents to deploy previews on [Prevu Cloud](https://prevu.cloud).

## Available Skills

### [`prevu-deploy`](./prevu-deploy/)

Deploy project previews to Prevu Cloud. Supports static sites, backend services (Python/Node/Go), Docker images, and fullstack apps with infrastructure dependencies.

## What is Prevu?

Prevu gives you instant, temporary preview URLs for any project. Deploy a frontend, backend, or fullstack app and get a live `*.prevu.page` link in seconds — with auto-provisioned databases, TLS, and auto-expiry.

**Life's too short, Prevu it first.**

## Usage

These skills are designed for AI coding agents (Codex, Claude Code, Cursor, etc.) to understand how to deploy projects to Prevu. Install as agent skills or reference the SKILL.md directly.
